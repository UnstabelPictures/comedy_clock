a basic HTML5 timer designed specifically to run McCurdy's Comedy Theatre's shows.  

FEATURES INCLUDE:
Timer that counts up
Clock that displays PCs time
Dark Minimalist theme (ideal for facing towards the stage)
Warn button that turns timer red
Flash button that flashes entire screen
timer change controls (revealed by "+" in the bottom left corner)

the comedy clock is always free and open-source thanks to McCurdy's Comedy Theatre and Humor Institute.  

But if you find it useful please consider donating to Kevin Krasko on paypal @hrexisvexed
